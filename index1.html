Overview

The World Wide Web is a wide-area hypermedia information retrieval initiative aiming to give universal access to a large universe of documents. This page is a visual re-creation of an early CERN informational page. It's intentionally minimal, with serif text, monospace navigation, and underlined blue links — the hallmark of early Web pages.

Design notes

The original design favored a single, narrow column for reading, a left-hand text navigation column, and content that used plain HTML elements without heavy graphics or modern layout techniques.

How to join

To contribute or run a local server, create simple HTML files and serve them from a web-accessible host. Early adopters commonly used plain text editors and FTP for upload.

History

The World Wide Web project began as an experiment in distributed information systems. This page pays tribute to the early style rather than reproducing any particular document word-for-word.
